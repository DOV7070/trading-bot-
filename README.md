# 🤖 Trading Bot (Binance) — RSI / MA Cross / MACD

Bot Python prêt à déployer sur **Railway** ou **Render**, avec **paper trading** sur le Binance Testnet, dashboard Flask, backtester intégré et gestion du risque (stop-loss / take-profit).

> ⚠️ **Avertissement risque** : aucune stratégie ne garantit le profit. Les performances passées ne préjugent pas des performances futures. **Toujours tester en paper trading** avant d'engager des fonds réels. Tu es seul responsable de tes ordres.
